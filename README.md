# truenas-charts

Charts catalog for TrueNAS Scale

In order to install the catalog.

- Section `Apps`
- Tab `Manage Catalog`
- Button `Add Catalog`
    - `Catalog Name` -> `jbgosselin` 
    - `Repository` -> `https://github.com/jbgosselin/truenas-charts`
    - `Preferred Trains` -> `charts`
    - `Branch` -> `main`
